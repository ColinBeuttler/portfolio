Technologies Used on This Site

Pure CSS

A very lightweight and responseive CSS framework that uses a very clean and minimalistic style. The website is the first time I have used Pure CSS, I picked it for its simplistic design that helped to add much needed subtle design changes to my site.

jQuery

A lightwieght and fast library used with Javascript. I have used jQuery is the past and have gotten acustomed to its ease. I especially like using it in conjunction with parallax.
